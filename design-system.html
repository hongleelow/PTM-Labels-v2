Guidelines

Usage Rules

Color Rules

  • ✓ Use primary-500 for buttons, links, and icons
  • ✓ Use primary-50 for light backgrounds and icon containers
  • ✓ Use gray-900 for headings, gray-500 for body text
  • ✓ Use gray-50 to alternate section backgrounds
  • ✗ Never use primary green for large background areas (use section--primary sparingly)
  • ✗ Never use gray-400 or lighter for body text (fails contrast)

Spacing Rules

  • ✓ Use space-lg (24px) between related elements
  • ✓ Use space-2xl (48px) between content blocks
  • ✓ Use .section (80px) between page sections
  • ✓ Alternate white / gray sections for rhythm
  • ✗ Never use less than 16px margin between text elements
  • ✗ Never stack two gray sections without a white section between
